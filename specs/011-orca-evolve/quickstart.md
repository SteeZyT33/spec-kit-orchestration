# Quickstart: Orca Evolve

## Goal

Track external ideas Orca still wants to adopt and map them into real Orca
work.

## Example Flow

1. Capture a Spex-derived idea as a harvest entry.
2. Mark it `direct-take`, `adapt-heavily`, `defer`, or `reject`.
3. Link the entry to an existing Orca spec or future feature slot.
4. Revisit the Evolve inventory to see which ideas remain open or deferred.
5. Update the entry when the mapped Orca work is implemented.

## Expected Operator Value

- less repeated rediscovery work
- clear trace from external pattern to Orca destination
- durable record of why Orca did or did not adopt an idea
